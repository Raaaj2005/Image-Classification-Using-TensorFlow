# 🖼️ Image Classification Using TensorFlow

## 📖 Overview
This project serves as a foundational exploration into computer vision and deep learning by building an image classification model from scratch. Using the classic MNIST dataset of handwritten digits, the notebook walks through the entire workflow—from understanding the raw image data structure and applying one-hot encoding, to designing, training, and evaluating a Neural Network using TensorFlow and Keras.

## 🎯 Objective
The primary objective is to grasp the core mechanics of Neural Networks and build a sequential model capable of accurately classifying images. By manually handling data pre-processing steps like unrolling features and normalization, and carefully selecting activation functions, the project aims to demystify how deep learning models learn from raw pixel data to make accurate predictions.

## 🛠️ Tech Stack & Tools
* **Language:** Python
* **Environment:** Jupyter Notebook
* **Deep Learning Framework:** TensorFlow / Keras
* **Data Manipulation & Visualization:** NumPy, Matplotlib

## 🧠 Key Learnings & Skills Demonstrated
* **Dataset Exploration:** Importing and analyzing the structure of the MNIST dataset to understand raw image examples.
* **Data Transformation:** Applying One-Hot Encoding to effectively format categorical labels for neural network processing.
* **Image Pre-processing:** Preparing image data by unrolling input features and normalizing using mean and standard deviation.
* **Neural Network Architecture:** Building a Sequential model in Keras, configuring hidden layers, and applying ReLU and Softmax activation functions.
* **Model Compilation & Training:** Defining optimizers and loss functions, fitting the model to the training data, and evaluating its accuracy.
* **Prediction Visualization:** Generating predictions on the unseen test set and visualizing the model's accuracy against the actual images.

## 🚀 Getting Started

### 📋 Prerequisites
To run this project, you will need:
* Python 3.x
* Jupyter Notebook or JupyterLab

### ⚙️ Installation & Setup

1. Clone the repository to your local machine:
   git clone https://github.com/Raaaj2005/Image-Classification-Using-TensorFlow.git

2. Navigate into the downloaded project folder:
   cd Image-Classification-Using-TensorFlow

3. Install the required dependencies:
   pip install tensorflow numpy matplotlib jupyter

4. Launch the Jupyter Notebook environment:
   jupyter notebook

5. Open the project's .ipynb file and execute the cells sequentially to preprocess the MNIST data and train the classifier.
